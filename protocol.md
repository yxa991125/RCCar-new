| 5 | short（寄存器原始值）低8位 | Y轴速度低8位 |
| 6 | short（寄存器原始值）高8位 | Z轴速度高8位 |
| 7 | short（寄存器原始值）低8位 | Z轴速度低8位 |
| 8 | short（寄存器原始值）高8位 | X轴加速度高8位 |
| 9 | short（寄存器原始值）低8位 | X轴加速度低8位 |
| 10 | short（寄存器原始值）高8位 | Y轴加速度高8位 |
| 11 | short（寄存器原始值）低8位 | Y轴加速度低8位 |
| 12 | short（寄存器原始值）高8位 | Z轴加速度高8位 |
| 13 | short（寄存器原始值）低8位 | Z轴加速度低8位 |
| 14 | short（寄存器原始值）高8位 | X轴角速度高8位 |
| 15 | short（寄存器原始值）低8位 | X轴角速度低8位 |
| 16 | short（寄存器原始值）高8位 | Y轴角速度高8位 |
| 17 | short（寄存器原始值）低8位 | Y轴角速度低8位 |
| 18 | short（寄存器原始值）高8位 | Z轴角速度高8位 |
| 19 | short（寄存器原始值）低8位 | Z轴角速度低8位 |
| 20 | short（单位 mV）高8位 | 电池电压高8位 |
| 21 | short（单位 mV）低8位 | 电池电压低8位 |
| 22 | uint8_t | BCC校验位 |
| 23 | uint8_t | 帧尾 `0x7D` |

## 协议 5：自动回充数据包（CAN）

- **名称**：自动回充数据包
- **方向**：S23A -> C50A
- **数据方向**：S23A自动回充装置 -> C50A主控制器
- **数据通信方式**：CAN通信
- **帧ID**：`0x182`
- **总长度**：8 字节

### 字节定义

| 字节 | 类型 | 含义 |
|---|---|---|
| 0 | short（单位 mm/s） | X轴目标速度高8位 |
| 1 | short（单位 mm/s） | X轴目标速度低8位 |
| 2 | short（单位 0.1V） | 充电桩测量电压高8位 |
| 3 | short（单位 0.1V） | 充电桩测量电压低8位 |
| 4 | short（单位 mm/s） | Z轴目标速度高8位 |
| 5 | short（单位 mm/s） | Z轴目标速度低8位 |
| 6 | uint8_t | 第0位表示充电状态，第1位表示红外接收情况 |
| 7 | 8位有符号数（单位 0.033mA） | 充电电流 |

## 协议 6：目标速度数据（CAN）

- **名称**：目标速度数据
- **方向（原图左上）**：C63A -> 伺服驱动器
- **数据方向（原图中上）**：C50A主控制器 -> 伺服驱动器
- **数据通信方式**：CAN通信
- **帧ID**：`0x601`
- **总长度**：8 字节

### 字节定义

| 字节 | 类型 | 含义 |
|---|---|---|
| 0 | uint8_t | 命令字（`0x23`） |
| 1 | uint32_t | 伺服控制器控制目标速度的地址（`0x60FF`） |
| 2 | uint32_t | 伺服控制器控制目标速度的地址（`0x60FF`） |
| 3 | uint8_t | 目标地址索引号（`0x03`） |
| 4 | short（单位 rpm） | 左轮速度高8位 |
| 5 | short（单位 rpm） | 左轮速度低8位 |
| 6 | short（单位 rpm） | 右轮速度高8位 |
| 7 | short（单位 rpm） | 右轮速度低8位 |

> 注：原表中 `1~2` 字节标注为 `uint32_t`，这里按原图保留。

## 协议 7：编码器数据（CAN）

- **名称**：编码器数据
- **方向（原图左上）**：伺服驱动器 -> C63A
- **数据方向（原图中上）**：伺服驱动器 -> C50A主控制器
- **数据通信方式**：CAN通信
- **帧ID**：`0x190 + 驱动id`
- **总长度**：8 字节

### 字节定义

| 字节范围 | 类型 | 含义 |
|---|---|---|
| 0..3 | int（单位 0.1 r/min） | 左轮电机测量的转速（低字节在前，高字节在后） |
| 4..7 | int（单位 0.1 r/min） | 右轮电机测量的转速（低字节在前，高字节在后） |
